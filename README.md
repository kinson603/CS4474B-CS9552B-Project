# CS4474B-CS9552B-Project (Group 16)
The GitHub repository for the group project of CS4474B/CS9552B: Human Computer Interaction

## Game executable
Location: Zombie Math Game/Build/Zombie Math Game.exe
Clicking it should start the game without any problems.

## Source Code
Location: Zombie Math Game/Assets/Scripts
The folder contains all the code needed to run the game, including the game manager, player/enemy controllers, etc. As the game is built with Unity, building from source code requires the Unity editor to open it. The editor version used for the game is **6000.3.10f1**.
